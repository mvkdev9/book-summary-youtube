# Book Summary YouTube Project

This repository contains 365 summaries of the most influential non-fiction books of all time, covering productivity, psychology, finance, leadership, relationships, and spirituality.

Each file includes:
1.  **A Long Detailed Summary** (optimized for blog posts).
2.  **A Long-Form YouTube Script** (approx. 10 minutes).
3.  **Short-Form Video Scripts** (for TikTok/Reels/Shorts).

## Project Goal
To provide high-quality, actionable content for content creators, learners, and readers who want to digest the world's best wisdom efficiently.

## Directory Structure
The files are named with a numbering system (001-365) followed by the Title and Author.

### Categories
*   **Productivity & Time Management** (e.g., *Atomic Habits*, *Deep Work*)
*   **Mindset, Psychology & Personal Growth** (e.g., *Mindset*, *Grit*)
*   **Relationships & Communication** (e.g., *Crucial Conversations*, *The 5 Love Languages*)
*   **Health, Wellness & Biohacking** (e.g., *Why We Sleep*, *Breath*)
*   **Wealth, Finance & Career** (e.g., *Rich Dad Poor Dad*, *Zero to One*)
*   **Philosophy, Spirituality & Life Lessons** (e.g., *Tao Te Ching*, *The Power of Now*)

## Usage
These scripts are templates. Feel free to adapt the tone, examples, and visuals to match your personal brand.
